Instructions for Updating the data
Option 1 – Import CSV
1. Download the template
2. Click on the template; provide a file name to save the CSV.
3. Update the data as per the headings provided in the CSV template.
Option 2 – Fill Data
1. Select "Fill details", enter all the required details .To add rows select "Add" and update the details. To delete rows select the row which is to be deleted and then click on "Delete".
2. Click on "Save".
Note:
1. All the imported files should only be in .CSV format.
2. To modify / Delete any of the records, Re-Import the new corrected .CSV file.
3. To avoid incorrect data, Taxpayer should use either "Option 1- Import CSV" or "Option 2-Fill Data".
