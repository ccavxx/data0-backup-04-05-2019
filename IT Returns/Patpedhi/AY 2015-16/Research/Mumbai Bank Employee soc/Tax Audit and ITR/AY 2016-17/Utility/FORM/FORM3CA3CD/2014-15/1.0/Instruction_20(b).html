Instructions for updating the data at Point No. 20 (b) of Form 3CD
Option 1 – "Import CSV"
1. Download the template through the link "CSV Template"and provide a file name to save the CSV file.
2. Update the data as per the headings provided in the CSV template
3. To upload the CSV file click on “Import CSV” and browse the filled CSV file
Key points to be considered while filling CSV file
1. The drop down values of “Nature of Fund” should be given as the codes mentioned below:
Drop down Code to be specified in CSV(case sensitive)
Provident Fund PROVIDENT
Superannuation Fund SUPERAN
Gratuity Fund GRATUITY
Any Fund set up under the provisions of ESI Act, 1948 PROV1948
Any other welfare fund OTHERWELFARE

2. “Date” field should be in the format “DD/MM/YYYY” or “DD-MM-YYYY”

Option 2 – Fill Data
1. Click on “Fill data”, to enter all the required details .To add rows click “Add” and update the details. To delete rows select the row which is to be deleted and then click on “Delete”.
2. Click on “Save” to save the filled data and go back to Clause 20(b)
Note: This option should be used if there is limited number of rows
