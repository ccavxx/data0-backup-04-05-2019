Instructions for filling Depreciation Schedule
Option 1 – Import CSV
1. Download the template through the link (Additions CSV Template / Deductions CSV Template)
2. Click on the template; provide a file name to save the CSV.
3. Update the data as per the headings provided in the CSV template.
Option 2 – Fill Data
1. Select "Fill data", enter all the required details .To add rows select "Add" and update the details. To delete rows select the row which is to be deleted and then click on "Delete".
2. Click on "Save" to go back to Depreciation Schedule.
Note: This option shall be used if there is limited number of additions/deletions data
Guidance for importing the data
In "Additions CSV Template", the data provided should comply with the below requirements:
a)"Date of Purchase" field should be in the format "DD/MM/YYYY" or "DD-MM-YYYY"
b) "Date Put to Use" field should be in the format "DD/MM/YYYY" or "DD-MM-YYYY"
c) "Purchase value" field should be numeric, non- negative and non- decimal.
d) "CENVAT" field should be numeric and non- decimal.
e) "Change in Rate of Exchange" field should be numeric and non- decimal.
f) "Subsidy/Grant" field should be numeric and non- decimal.
f) "Total Value of Purchases" field should be numeric, non- negative and non-decimal.
In "Deductions CSV Template", the data provided should comply with the below requirements:
a)"Date of Sale etc.," field should be in the format "DD/MM/YYYY" or "DD-MM-YYYY"
b) "Amount" field should be numeric, non- negative and non- decimal.
Note:
1. All the imported files should only be in .CSV format.
2. To Modify / Delete any of the records, Re-Import the new corrected .CSV file.
3. To avoid incorrect data, Taxpayer should use either "Option 1- Import CSV" or "Option 2-Fill Data" of the above for a particular block of assets.
